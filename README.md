# Later

Later is a visual social media scheduling and influencer marketing platform that enables brands, agencies, and creators to plan, schedule, and publish content across Instagram, TikTok, Facebook, Pinterest, LinkedIn, Threads, and YouTube.

## Platform Features

- Visual drag-and-drop content calendar
- Media library for organizing images and videos
- Multi-platform scheduling (Instagram, TikTok, Facebook, Pinterest, LinkedIn, Threads, YouTube)
- Link in Bio tools for driving social commerce
- Analytics and performance insights
- Social inbox for managing comments and messages
- Content approval workflows for team collaboration
- AI-powered best time to post recommendations
- Competitive benchmarking and brand health monitoring
- Influencer marketing campaign management
- Creator network and affiliate programs (Mavely)

## Supported Platforms

- Instagram
- TikTok
- Facebook
- Pinterest
- LinkedIn
- Threads
- YouTube Shorts
